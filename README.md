# PHP MVC Architecture Demo

PHP MVC Architecture Demo is a framework-free PHP project created to practice Model-View-Controller separation, public entry-point structure, and classic application layering without relying on Laravel or another full-stack framework.

## Highlights

- Pure PHP application structure
- MVC-style separation between application logic and public entry point
- Useful reference for understanding framework fundamentals
- Lightweight repository for architecture practice and interview discussion

## Project Structure

- `app/`: application logic and internal structure
- `public/`: web-accessible entry point and assets
- `.htaccess`: routing or web-server rewrite support

## Getting Started

Serve the `public/` directory through a local PHP-capable web server such as Apache, Nginx, or the PHP built-in server configured for a public document root.

## Notes

This repository is an architecture-learning example rather than a production-ready application.
